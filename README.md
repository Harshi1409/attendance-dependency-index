# 📊 Attendance Dependency Index (INT375 Project)

## 🎓 Course

INT375 – Data Science Toolbox

## 📌 Project Description

This project analyzes the relationship between student attendance and performance using real-world data.

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📈 Techniques Applied

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Correlation Analysis
* Hypothesis Testing
* Linear Regression

## 📊 Key Result

The analysis showed a weak correlation between attendance and performance.
